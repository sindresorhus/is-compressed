# is-compressed

> Check if a file path is a compressed file


## Install

```
$ npm install is-compressed
```


## Usage

```js
const isCompressed = require('is-compressed');

isCompressed('source/unicorn.zip');
//=> true

isCompressed('source/unicorn.txt');
//=> false
```
